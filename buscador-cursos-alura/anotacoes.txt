##dom Crawler percorre o HTML
Um pacote para fazer a requisição HTTP
packagist.org é onde o composer busca os pacotes, mas pode buscar de outros lugares
Para instalar uma nova dependencia devemos adicionala no composer.json dentro de require e executar o composer update ou fazer apenas o composer require.
o autoload fica disponível automaticamente na pasta vendor
Para o código continuar funcionando como no curso foi alterado dentro do arquivo Client.php->configureDefaults a variavel 'verify' para false
Para deixar enviar request para fora por questoes de cacert.pem resolve o erro CURLE_PEER_FAILED_VERIFICATION (60) fazer apenas local
Como foi criado uma classe "Buscador" ela tem que ser importada separado, pois o composer não consegue importar ela pelo autoload
